# news-test

To run server:
1. install node js https://nodejs.org/en
2. install globally json-server: npm install -g json-server
3. go to server directory: cd ./mocked_server
4. redirect port: adb reverse tcp:4000 tcp:4000
For more info: https://stackoverflow.com/questions/49855754/unable-to-make-calls-to-localhost-using-flutter-random-port-being-assigned-to-h/60653912#60653912)https://stackoverflow.com/questions/49855754/unable-to-make-calls-to-localhost-using-flutter-random-port-being-assigned-to-h/60653912#60653912
5. run the following command: bash start-server.sh
